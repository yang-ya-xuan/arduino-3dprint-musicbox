# Arduino 與 3D列印- 智慧互動音樂盒

使用 Arduino UNO 開發板與 3D 列印機構設計所製作的智慧互動音樂盒。系統內建生日快樂、小星星、致愛麗絲多首經典曲目，並透過光敏感測器即時偵測蓋子開關環境光線狀態，實現無源蜂鳴器音樂播放，自動化即時靜音中斷以及隨機曲目抽選，並且使 9組LED燈能隨著特定音符頻率的變換，同步切換對應的燈泡亮滅，打造動態影音特效；此外，內部硬體電路皆由個人手工焊接完成，整體使用SOLIDWORKS建模後列印，完美收納內部硬體電路。
